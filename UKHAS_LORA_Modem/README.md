# UKHAS_LORA_Modem

## Commands
*  F - Freqency (float)
*  P - Power Output
*  I - ID 
*  W - Sync Word
*  L - Preamble Length
*  C - Custom settings (SF, CR, Bandwidth as a float)
*  M - Mode (currently on 5)
*  D - Debug
*  R - CRC Toggle On/Off
*  K - Low Data Rate On/Off
*  U - UKHAS Packet format On/Off
*  V - Version
*  S - Output as String or Binary
